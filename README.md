# HABIT TRACKER
Un tracker de hábitos simple, con racha de 5 días y estilos simples.

## Aprendizajes
- Estructura de carpetas simple
- Uso de .map para el render 
- UseEffect para LocalStorage

## Dificultades
- Separación de componentes presentacional/contenedor
- Actualizar el estado en la función handleToggleDay
- Hacer una fila por día con .map
- Implementar el onClick en las filas

## Tecnologías
- HTML
- CSS
- JAVASCRIPT
- REACT
